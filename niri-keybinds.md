# Niri Keybinds

Optimized for 4x6 compact split keyboard (HJKL-based navigation, no arrow/Page/Home/End keys required).

See also: [DMS Keybinds](dms-keybinds.md) for shell-specific bindings (launcher, clipboard, audio, brightness).

## System & Overview

| Keybind | Action |
|---------|--------|
| Mod+Tab | Toggle overview |
| Mod+Shift+/ | Show hotkey overlay |
| Mod+Shift+E | Quit Niri |

## Navigation Pattern Summary

All directional navigation uses HJKL (vim-style):

| Action | Left | Down | Up | Right |
|--------|------|------|-----|-------|
| Focus | Mod+H | Mod+J | Mod+K | Mod+L |
| Move | Mod+Shift+H | Mod+Shift+J | Mod+Shift+K | Mod+Shift+L |
| Monitor Focus | Mod+Ctrl+H | Mod+Ctrl+J | Mod+Ctrl+K | Mod+Ctrl+L |
| Monitor Move | Mod+Shift+Ctrl+H | Mod+Shift+Ctrl+J | Mod+Shift+Ctrl+K | Mod+Shift+Ctrl+L |
